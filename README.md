# タイトル：GrapPull

## ・どういったゲーム？
グラップガン（狙った物体に向かって飛ぶことができる銃）とプルガン（狙った物体を引き寄せることができる銃）を駆使してゴールを目指すゲーム。

操作方法
  ・キーボード&マウス
    WASDキーで移動、Spaceキーでジャンプ。
    左クリック - 狙った物体に向かって飛ぶ（グラップガン）
    右クリック - 狙った物体を引き寄せる  （プルガン）
    
[ゲームの説明動画](https://youtu.be/8TCSbDG72JU)
